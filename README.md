Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.
The game allows users to play against the computer with real-time score updates and result display.

🚀 Features

🎯 Play against the computer

🔄 Random computer choice generation

📊 Live score tracking

🎨 Dynamic result message with color indication

⚡ Fast and responsive gameplay

🛠️ Technologies Used

HTML – Structure of the game

CSS – Styling and layout

JavaScript – Game logic and interactivity

🎮 How to Play

Click on Rock, Paper, or Scissors.

The computer will generate a random choice.

The winner is decided based on the rules:

Rock beats Scissors

Paper beats Rock

Scissors beats Paper

Scores update automatically after each round.
